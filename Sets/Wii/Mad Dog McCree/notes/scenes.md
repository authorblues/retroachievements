# MenuGrid Values

MenuGrid is 0x011b in the Scene ID, but different frame values

## Main Menu
- 153, 208, 226 = Nintendo Cards

- 10 = MD1 Difficulty Select
- 18 = MD1 Player Select
- 34 = MD2 Difficulty Select
- 42 = MD2 Player Select
- 58 = LBH Difficulty Select
- 66 = LBH Player Select

- 124 = Scoreboard Name Entry

## MD1
- 2 = Area Select (Corral, Saloon, Sheriff, Bank)
- 100 = Area Select Sign (Prospector, Mine, Hide Out)
  
- 116 = High Scores
- 221-225 = Credits

## MD2
- 108 = Choose a Guide
- 132 = High Scores
- 197-207 = Credits

## LBH
- 140 = High Scores
- 173-186 = Credits
